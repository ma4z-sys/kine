# Kine

Run Android APKs on Linux, similar to how Wine runs Windows .exe files.

## Installation

git clone https://github.com/ma4z-sys/kine.git
cd kine
sudo bash install.sh

## Setup

Initialize a device:

kine init pixel5

Replace "pixel5" with any supported Android model.

## Usage

Run an APK:

kine launch your-app.apk

Example:

kine launch spookygooning.apk

## Notes

- App compatibility may vary
- Performance depends on your system
